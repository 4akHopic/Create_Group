Системні вимоги для максимально стабільної роботи:
Операційна система(ОС): Windows (7,8,8.1,10) або MacOS;
Web-браузер: Mozilla Firefox(версії 65.0.2 і вище) Google Chrome(не нижче 58 версії);
Стабільне підключення до інтернет мережі;

Управління додатком:
1. Користувач може створювати групи.
2. Додавати та видаляти команди у та з групи (якщо вже є результат матчу команди - видалення - заборонено).
3. Генерувати матчі у групі.
4. Зміна результати конкретного матчу.
5. Переглядати дані по групам (лист груп, лист команд у групах, лист матчів з результатами) зберігаються і доступні після перезавантаження сторінки.

Додаток розроблений скриптовою мовою програмування PHP(Personal Home Page Tools), за допомогою текстового редактора коду Atom. В якості локального веб сервера використовувався MAMP.

Інформація про ліцензійне використання знаходиться у файлі LICENSE.
